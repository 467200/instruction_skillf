### ПЕРВЫЕ НЕСКОЛЬКО ШАГОВ
Чтобы начать работать с **GIT**, его надо установить на комьютере. Выбираем самый популярный **GIT** hosting - ***GitHub***. Все, что необходимо для установки находится [здесь](https:/git-scm.com/downloads).

Далее необходимо произвести настройки, чтобы при создании [коммита](dictionary.md) указывался автор, кто его создал.
Для этого надо открыть терминал (_Linux и MacOS_) или консоль (_Windows_) и ввести следующие команды:<br/>
```bash=
git config --global user.name "your name" 
git config --global user.email "your_email@email.com"
```
Следующим шагом будет создание [репозитория](dictionary.md). В правом верхнем углу любой страницы откройте раскрывающееся меню **+** и выберите 
```bash=
New repository
```
![new repository](https://habrastorage.org/r/w1560/getpro/habr/upload_files/318/b86/cb0/318b86cb0768ee96f7757dadeb120a93.png)

 
1.Repository name: имя репозитория.

2.Description: описание репозитория.

3.Тип репозитория: 
   * Public (публичный) 
   * Private (приватный)

4.Галку на «Создать README файл». В этом файле в формате ***MarkDown*** описывают проект или прочую документацию. Именно содержимое этого файла можно увидеть, когда заходим на главную страницу репозитория. 

5.Если известно, на каком языке будет проект, можем добавить шаблон *.gitignore* для этого языка. 

6.Выбор типа лицензии. В лицензии оговариваются права на проект. Стоит обратить внимание на BSD 3 или MIT, так как они предоставляют хороший баланс прав и ответственности.

7.По умолчанию имя основной ветки в GitHub носит имя main.

8.Нажать кнопку «**Create repository**».
